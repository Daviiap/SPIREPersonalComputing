# SPIRE for Personal Computing

This repository provides the reference implementation and proof-of-concept for SPIRE for Personal Computing. This work was originally presented and published in the 14th Latin-American Symposium on Dependable and Secure Computing (LADC 2025).

## Publication & Citation
If you use this code or refer the work in your research, please cite our paper:
